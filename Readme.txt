This is a   simple python crawler to crawl travelyaari.com for bus schedule



# Make sure you install python with setuptools.

steps:

1. First run Setup.py
 
$ sudo python setup.py develop



After that 



2. run crawler.py


$ python crawler.py


Then input source city,date and destination without space like this :

$ CityFromdelhi
$ departuredate02-06-2015
$ CityToMumbai
